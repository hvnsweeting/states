Example::

  sentry:
    db:
      password: userpass
    workers: 2
    hostnames:
      - sentry.example.com
    django_key: randomstring
    initial_admin_user:
      username: joe
      password: test
      email: joe@test.com
    monitoring_timeout: 3

sentry:clean_days
~~~~~~~~~~~~~~~~~

:doc:`index` runs a daily crob job to clean up old data. Data that live longer
than this pillar value (in days) will be deleted.

Default: do not delete old data (``False``).
